Tilecache API’s¶

getSimpleTile¶

GET (string: server_name)/ocptilecache/tilecache/(string: token_name)/(string: channel_name)/(string: slice_type)/(int: time)/(int: zvalue)/(int: ytile)_(int: xtile)_(int: resolution).png¶
Synopsis:

Get a 512x512 tile from the tilecache
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is ocp.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP. Optional. If missing will use default channel for the token.
  • slice_type (string) – Type of Slice cutout. Can be xy/yz/xz
  • time (int) – Time value. Optional. Only possible in timeseries datasets.
  • zvalue (int) – Zslice value.
  • ytile (int) – Y-Tile value. Each tile is 512x512.
  • xtile (int) – X-Tile value. Each tile is 512x512.
  • resolution (int) – Resolution value.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

getMcfcTile¶

GET (string: server_name)/ocptilecache/tilecache/mcfc/(string: token_name)/(string: channel_name):(string: color_name)/(string: slice_type)/(int: time)/(int: zvalue)/(int: ytile)_(int: xtile)_(int: resolution).png¶
Synopsis:

Get a 512x512 tile from the tilecache
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is ocp.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP. Optional. If missing will use default channel for the token.
  • color_name (string) – Color Name. Can be ‘C/M/Y/R/G/B’. Optional If Missing will default to “CMYRGB”.
  • slice_type (string) – Type of Slice cutout. Can be xy/yz/xz
  • time (int) – Time value. Optional. Only possible in timeseries datasets.
  • zvalue (int) – Zslice value.
  • ytile (int) – Y-Tile value. Each tile is 512x512.
  • xtile (int) – X-Tile value. Each tile is 512x512.
  • resolution (int) – Resolution value.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format
